我的自我介绍

如果说您能看到这个页面，那我真的非常非常开心，因为这个网站我并没有和任何人提起过，找到这个网站只有从我的简历上找到。我大胆的猜测一下，您可能是HR，也可能是我未来的同事或者我的领导~

最初是想搭建一个网站，记录一下自己的学习笔记，思来想去，那些笔记多半还是复制粘贴的文字以及一些截图，与其让您看这些内容，不如乘此好好介绍一下我自己，可以更立体的让您认识我，理解我以下的一些问题：为什么我本科化学要跨考计算机？明明硕士方向是计算机视觉目标跟踪，为什么学 Java ？为什么要找Java后端开发？一战专业课第一仍然选择跨考的压力大不大？为什么一战不考计算机？如何学习java的？

​ 我将按照时间线来介绍一些我记忆深刻的事情，会从我的小学一直介绍到现在：

​ 在我小学五年级时候，家里买了第一台惠普的电脑，那时候2008年左右，在我们那农村已经算是比较稀有的设备，因为父母开始了收取电话费的工作（那时候手机并没有很普及，电话还是非常主流的通信工具）才因业务需要买了一台电脑。那时候我还很小，最开始只是在电脑上看动画片，大概半年后慢慢学会了打游戏，CF、QQ飞车、DNF等等，因为游戏，我对计算机产成了第一波热爱，我想很多孩子和我一样。

​ 大概到了我初二的时候，机缘巧合下，认识一个网友，他说可以教我一些黑客技术，但是需要50块钱的拜师费，对于那时候的我，非常崇拜这样的黑客技术，觉得特别的牛~，我存了三星期的钱，最后用QB的形式充给了那个网友，当然不出意外，他是个骗子，或者有技术但是不想教我。虽然他是一个骗子，但是他好像还是懂一些东西，我依稀的记得他和我说用易语言去写一个程序什么的，然后在输入框里写上链接，别人自己输入自己的QQ账号密码后，会把输入内容转发到我的QQ邮箱中~ 那时候的互联网并没有现在发达，我想搜索的内容在百度都是没有的，相关技术也是几乎没有，每次搜完百度都是“期待您的回答~”。。。强烈的兴趣下，自己慢慢摸索，下载了易语言，也花了几天自己写一个软件模样的界面，一步步的尝试和探索，最后，我成功了！我并没有去骗人，只是自己单机测试，我随便输入什么内容，确实可以发送到我自己的 QQ 邮箱中。那一刻，我就觉得我是个黑客，我特别享受那种自己写代码，做软件的感觉（虽然也没写啥），内心得到极大的满足，我太爱计算机写代码的感觉，我觉得这是一件很帅的事情。因为这个事情的鼓励，我在这方向是愈发的入迷，我努力的在各种黑客，红客网站中学一些我感兴趣的技术，直到有一天，发现有一个bug，大概目的就是可以用一个月的钱去开通好多个月的会员业务（阿弥陀佛，那时候真的太小了，现在绝对不会做这样的事，其实是一个bug，现在早就解决），我学会后，开始去身边同学或者网上的群聊里打广告，开始通过计算机做起了小本生意，赚点小钱，然后再去学更多网络技术，然后赚更多的钱，反复迭代。最高记录应该是一周赚了2000块吧，这可是我半年多的生活费！这时候应该是我热爱计算机的高潮，我恨不得天天在电脑面前，去学更多的技术、去敲更多的代码、去赚更多的生活费。我内心也坚定的认为，以后我要做自己热爱的事情，我要从事计算机相关的工作，从此心中种下了一颗种子。

​ 中考还算顺利，考试县里最好的高中，但是离家非常的远，几个月才能回去一次，也因此高中三年与互联网断开了联系，没有机会去接触电脑。

​ 转眼间，高考结束，考出了高中最低的一次分数，即使我难以接受（已经联系好复读班的班主任），但是考虑家庭的种种原因，最终不得不接受这个事实，只好去上大学。我记得很清楚，填志愿的前一天晚上，我和父亲在床头拿着志愿手册，挑了无数的学校，学校已记不清，但我记得当初选择的专业全部都是计算机相关专业。也憧憬着自己上大学好好学计算机场景。录取信息出来了，并没有按照我想的一样，录取计算机的专业，而是给我录取了“应用化学”；

​ 上了大学，一切顺利，我早早的进实验室开始锻炼自己的动手能力，本科导师也很器重我，国家重点基金的项目也让我做其比较核心的部分，我因此进步也巨大，很快到了该考研的时间，但是毕竟学校是双非，我并没有说学校哪里不好，只是在那种氛围下，师兄师姐他们好像丧失了一种斗志，更多的是“认命”，我记得很清楚，他们认真的劝我“考研真的很难，不要去报重点，能考上个普通的研究生就可以了，我们双非考研也会被歧视的”，唉，现在想起也觉得有一些辛酸！最后我选择去考合肥工业大学，在我们安徽，他是重点211。实话说，我完全没有考虑过考研考计算机这件事，在我当时的认知中，考研是极其困难的事情，已经被困在信息壁障中。考研成绩出分的时候，我还是挺开心的，394分，因为是理学是不需要考数学的，两门专业课，135,137，专业第一。说来也巧，正值疫情最严重的时候，我在家里足不出户，每天不能出门，愈发感慨世界并不和平，人生说短也短，如果短暂的人生每天不能做自己喜欢的事情，那岂不是人生更加的无彩。终究心中的种子还是发芽了，我想学计算机，我就是想学计算机，从小到大从未改变。艰难说服父母后，决定放弃去读研，去二战追求自己喜欢的事情，毕竟农村嘛，我们那边教育还是有限的，考上研那已经是非常稀有的事情，还是经常能听到别人讨论我“作”的行为，压力其实还是蛮大的，并不是害怕考不上，而是会让别人“作”的说法成真。二战对我来说是全部从零开始的，因为理学不考数学，数学一我需要从基础开始学习，同样，计算机的专业课，我也全部需要从零基础开始。

​ 庆幸二战还算顺利，最后专业课133分专业课前五，总分367分，成功上岸西安电子科技大学，也成功进入了学校比较有名的人工智能团队（我并不知道我想做的方向具体是什么，我只记得初中时候的感觉，我想要那种感觉，但我对应不起来具体的方向）。

​ 读研期间，科研方向是计算机视觉-目标跟踪方向，应用多用于军事相关以及自动驾驶等领域，我的性格比较轴，就是不服输那种状态（数学题越做不出来，我就越要做到底那种，一题做一天），由于基础较差，刚开始，确实极其痛苦，一个个的ddl的到来，我都是付出更多的时间去完成，慢慢对人工智能也有了一个较深入的认知，也知道每天我的工作需要做什么，应该怎么去做，逐渐发现，至少我做的层面，我做的方向，是需要依靠大量的GPU算力，以及各种消融调参，重复实验。这种调参的感觉我并不喜欢，同样，由于深度学习的不可解释性，很多现象并没法去合理的解释，我觉得很大程度依靠一个人的“运气”。虽然我可以做好，但是深度学习很明显不是我想追求的事情。一直以来往届的师兄师姐找工作也只有两种，一种算法深度学习，一种就是C++偏硬件的那种。我也尝试的去学习了下C++，去看一看师兄们的工作，我确实是非常不感兴趣偏硬件的业务。在一次次尝试中，尝试了下java，我狂喜，这就是我想要的感觉，我跨考计算机就是为了这个方向的感觉，去做一些业务开发，可以展示到网页或者软件上，随便做个小项目就可以立马在浏览器上访问到效果，我太喜欢这种感觉，非常开心，我找到了自己想做的方向。

​ 基本上，从2022年3月逐渐开始学习java基础，六月基本上开始逐渐增大java每天的时间占比，尽量是在工作和毕业之间达到一个平衡。学习的途径很多，主要通过一线大厂工作的朋友去咨询，现在java中，哪些技术栈是比较重要的，哪些板块是比较重要的，尽量是面向工作去学习，优先去学习工作中息息相关的技术。当然也会在B站，慕课，牛客，各种论坛等通过看视频，看文章，自己去实践操作，记笔记等方式去学习，在保证学习质量的情况下，尽量多的去学习了解更多的知识，扩大自己的知识面。

​ 导师不放实习，只好花更多的时间去看更多的源码，认真夯实基础，提前批，秋招会逐渐开启，希望自己找工作顺利，也非常感谢您的阅读。

​ （我已经尽量的删减我所写的内容，但是确实有太多想去说的，非常感谢您的阅读，我已准备好在面试接受拷打！！！）
